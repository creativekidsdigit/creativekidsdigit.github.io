# Pinterest Deployment Package — All 10 Blog Posts

**Goal:** A copy-paste-ready Pinterest engine that drives traffic to the existing blog cluster, captures emails, and converts to Payhip sales.

**Output:** 50 pins (10 posts × 5 pins each), 30-day posting schedule, 12 boards, full keyword strategy, internal linking map, product funnel map.

---

## How to Use This Document

For each pin you create:
1. Open Canva, use a 1000×1500 px vertical template
2. Apply the **Common Design Brief** for that post (colors, fonts, vibe)
3. Place the **Text Overlay** copy on the image (large, readable, max 2 lines)
4. Save with the **Filename** suggested
5. Upload to Pinterest with the **Pin Title** + **Description**
6. Set **Destination URL** to the blog post (NEVER directly to Payhip)
7. Pin to the **Primary Board** + 1-2 **Secondary Boards**
8. Repeat 5 times per post (5 different hooks = 5 different pins for the same article)

---

## Brand Design System (applies to all pins)

| Element | Spec |
|---------|------|
| Pin size | 1000 × 1500 px (vertical 2:3 ratio) |
| Logo | Bottom-right corner, small, "Fearless Creations" |
| Brand colors | Teal `#7FBFBF`, Lavender `#9B7FBF`, Cream `#FAF6EE`, Peach `#E8A87C`, Soft Yellow `#E8C87A` |
| Title font | Fraunces (serif, bold) — also OK: Playfair Display, Cormorant |
| Body font | Inter (sans-serif) — also OK: Lato, Montserrat |
| Title size | 60-90 px (must read at thumbnail size) |
| Subtitle size | 28-40 px |
| Image style | Soft, warm, slightly muted; real parents/kids OK if no faces visible; flat illustrations also work |
| Avoid | Neon colors, hard edges, busy backgrounds, stock photo "perfection" |
