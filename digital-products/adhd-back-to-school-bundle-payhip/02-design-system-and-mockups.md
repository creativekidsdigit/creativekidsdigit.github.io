# 02 — Design System & Mockups

**Doc purpose:** Canva master setup, page-by-page design prompts, 5 mockup briefs, and the production checklist for the ADHD Back-to-School Bundle.

---

## A. Canva master file setup (build once, reuse for every product)

### A.1 Document setup

- Format: US Letter, 8.5 × 11 in
- Bleed: 0.125 in
- Safe area: 0.75 in margins
- Color profile: CMYK-safe palette (works on screen + print)
- Inner gutter: 0.5 in (matters for KDP adaptation later — `03-payhip-listing-and-platforms.md` §3)

### A.2 Brand palette (hex)

| Token | Hex | Use |
|---|---|---|
| Sage | `#A8C5A8` | Primary accent, headers, calm tone |
| Cream | `#FAF6EE` | Background, neutral cards |
| Terracotta | `#D49A7C` | Secondary accent, "stop / no" cues |
| Butter | `#E8D88A` | Highlight callouts, CTAs |
| Charcoal | `#3A3A3A` | Body text |
| Gray rule | `#E0DCD2` | Fillable lines, dividers |

### A.3 Typography

| Use | Font | Weights |
|---|---|---|
| Headings (display) | Fraunces | 600 / 700 |
| Body | Inter | 400 / 600 |
| Kid-facing accent | Caveat | 500 |

> Two type sizes max per page. No more than two font families on any page.

### A.4 Reusable components to build first (in this order)

1. **Page header bar** — sage strip with cream title block, page number top-right.
2. **Page footer** — small brand mark + page number bottom-center, in 9pt charcoal at 60% opacity.
3. **4 callout box styles** — sage tint, terracotta tint, butter highlight, cream framed.
4. **Checkbox row component** — square box + 12pt Inter line + optional icon.
5. **Fillable line component** — 0.5pt gray rule, 0.5-in left/right padding.
6. **Tracker grid component** — sage header row, alternating cream/white cells, gray rules.
7. **Cut-line + scissors pattern** — for cards meant to be cut and laminated.
8. **Cover frame template** — sage full-bleed + cream centered panel.
9. **Letter template** — cream paper + sage thin border, Fraunces italic body.
10. **Icon library** — 24 simple line icons in one weight.

> **Rule:** If a component is on more than one page, it lives in the master library. Never re-design it.

## B. Page-by-page design prompts

Pulls content from `01-bundle-content.md` verbatim. Each prompt is structured the same way so a solo creator can move through 14 pages in 1–2 working days.

### B.1 Cover (Page 1)

> Calm, premium printable bundle cover. Title: "The ADHD Back-to-School Bundle." Subtitle: "11 Printables for an ADHD-Friendly School Year — Ages 6–16." Background: full-bleed sage `#A8C5A8`. Centered cream `#FAF6EE` framed panel taking 65% of the page. Title in Fraunces 700, 60pt, charcoal. Subtitle in Inter 400, 22pt. Tiny terracotta accent line below the title. Bottom of cream panel: a row of 5 simple line icons (sun, moon, backpack, star, heart). No clipart kids. No stock photos. Footer in Inter 400, 11pt: "Fearless Creations · creativekidsdigit.github.io".

### B.2 How to Use This Bundle (Page 2)

> Two-column instructional layout. Left column (60%): a vertical numbered timeline (1–5) showing the printing order, each step with a small icon and 1-line action. Right column (40%): butter-yellow `#E8D88A` rounded callout titled "One rule" with the 2-sentence rule from the content doc. Header: "How to Use This Bundle."

### B.3 Morning Routine Chart (Page 3)

> Kid-facing chart, single page, 2×4 grid taking 80% of the page. Each tile: cream background, sage border, large numeral 1–8 in Fraunces 700, 32pt, simple icon (sun, toothbrush, water glass, T-shirt, plate, lunchbox, backpack, door), and 2–4 word label in Caveat 18pt. Header at top in Fraunces 700, 36pt: "My Morning." Below the grid: small fillable line "Today's bus / leave time: ____" and a 1-sentence parent note in italic.

### B.4 Bedtime Routine Chart (Page 4)

> Vertical 5-step timeline with icons (snack, bath, toothbrush, book, moon). Each step in a numbered row with cream background and sage rounded badge. Below the timeline: two side-by-side mini-cards — sage tinted "YES" with checkmarks (5 bullets), terracotta tinted "NO" with crosses (5 bullets). Butter-yellow callout strip at the bottom with the 2-sentence parent note. Header: "My Bedtime."

### B.5 Homework Planner (Page 5)

> Single-day fillable grid. Top header line: "Date: ____   Mood (1–5): ____" in Inter 600. 5 homework blocks alternating with "Break" rows. Each block has columns for Subject, What I'm doing, Time set, Done?. Sage header row, cream alternating fills, gray rule lines. Below the grid: sage tinted "End-of-session reflection" callout with 3 lined prompts in Caveat 14pt. Bottom: small italic parent line. Header: "Homework Planner — One Day."

### B.6 Weekly Planner — Page A (grid) (Page 6)

> Wide week-at-a-glance grid taking 80% of the page. 8 columns (label + Mon–Sun), 5 rows (Top priority, School/activities, Homework due, Mood 1–5, Win of the day). Sage header row, cream alternating cells, light gray rules. Below the grid: a single Caveat 14pt line for "Three things I need this week: ____." Header: "This Week." Subhead: "Week of: ____."

### B.7 Weekly Planner — Page B (reflection) (Page 7)

> Two halves divided by a sage horizontal rule. Top half: 5 kid-facing prompts in Caveat 16pt with dotted writing lines. Bottom half: 3 parent-facing prompts in Inter 400, 12pt inside a cream tinted box. Header: "Look Back, Look Ahead."

### B.8 Backpack Checklist (Page 8)

> Two-column layout. Left column sage tinted "GOING TO SCHOOL" with 10 checkbox items + small icons. Right column terracotta tinted "GOING HOME" with 7 checkbox items + small icons. Header at the top: "My Backpack." Subhead: "Two lists. One for school. One for home." Small italic parent note at the bottom.

### B.9 School Supply Checklist (Page 9)

> Five stacked tinted cards (alternating sage / cream / terracotta tints). Each card has: icon, category title in Fraunces 600, 14pt (WRITING TOOLS, ORGANIZATION, FOCUS TOOLS, SENSORY TOOLBOX, HOME-OFFICE FOR HOMEWORK), and 5–7 checkbox items in Inter 400, 11pt. Butter-yellow strip at the bottom with the 1-sentence parent note. Header: "ADHD-Friendly School Supply Checklist."

### B.10 Focus Tracker (Page 10)

> Wide tracker grid in the top 60% of the page. 6 columns (Block #, Time, Worked on, Focused 1–5, Distractions, Break). 5 rows. Sage header row. Below the grid: sage tinted reflection callout with 5 lined prompts in Caveat 14pt. Italic parent note at the bottom. Header: "Focus Tracker." Subhead: "Track 5 focus blocks. See what time of day works best."

### B.11 Reward System Chart (Page 11)

> Three vertical sections. Section A (top): 3-row goals table with cream cells. Section B (middle): 7-day star grid with sage header row. Section C (bottom): 5-row reward menu in butter-yellow tinted card. Italic parent note at the bottom. Header: "My Reward Chart."

### B.12 Emotion Regulation — Page A (feelings + body) (Page 12)

> Top third: large feelings-word cloud in Caveat 18pt — kid circles words. Middle third: simple body-outline silhouette (line-art only, no facial features) for coloring. Bottom third: 1–5 rating scale with a soft sage→terracotta gradient bar. Header: "What's Going On Inside Me?" in Fraunces 700, 36pt — kid-friendly large.

### B.13 Emotion Regulation — Page B (coping menu) (Page 13)

> 3×4 grid of icon tiles. Each tile: cream background, sage border, numeral 1–12 in Fraunces 600, 18pt, simple icon (breath, wall, water drop, fist, foot, music note, plate, pencil, hug, chair, speech bubble, teardrop), and 2–6 word instruction in Caveat 14pt. Footer line in Caveat 18pt centered: "Big feelings are not emergencies. They are weather. Weather always changes." Header: "What I Can Do Right Now."

### B.14 Parent–Teacher Log (Page 14)

> Header section with 3 small fillable lines (Child / Year / Teacher). Wide 5-column log table taking 65% of the page (Date, Method, Topic, Action, Follow-up date). 10 empty rows. Sage header row, light gray rules. Below: sage tinted box with 4 quarterly questions in Inter 400, 12pt. Italic parent note at the bottom. Header: "Parent–Teacher Log."

### B.15 Goal-Setting — Page A (kid) (Page 15)

> Kid-facing prompts in Caveat 16pt with big lined writing space (3 long prompts at the top, then "My goal in one sentence" in a butter-yellow callout box, then 3 numbered "signs" lines and 2 "team" lines). Page is 60% whitespace. Header: "My Goals" in Fraunces 700, 36pt — kid-friendly.

### B.16 Goal-Setting — Page B (parent) (Page 16)

> Parent-facing prompts in Inter 400, 12pt with handwritten lines. 5 prompts in the top half. "One rule" in a sage tinted callout in the middle. "Permission slip" in butter-yellow at the bottom, framed like a tear-off. Header: "Supporting My Child's Goal."

### B.17 Closing Letter (Page 17)

> Single-page letter on cream `#FAF6EE` background. Center-justified title in Fraunces 700, 32pt: "A Letter to the Parent." Body in Fraunces 500 italic, 16pt, line-height 1.7, max line length 60 characters. Small terracotta heart icon centered above the title. Signature in Inter 600, 12pt: "— Fearless Creations." Generous whitespace.

> Note on numbering: 17 final laid-out pages including the 2-page worksheets (Weekly Planner, Emotion Regulation, Goal-Setting). The bundle is described to buyers as "11 printables" because that's what they're buying — the count of worksheets, not the count of pages.

**Use:** Listing image #5 (the "what's inside" image). Pinterest pin variant — high-save format.

## D. Production checklist (run before exporting the final PDF)

- [ ] Cover, How-to-Use, all 11 printables, and Closing Letter present (17 designed pages).
- [ ] Page numbers in footer are correct.
- [ ] No leftover Lorem Ipsum / placeholder text.
- [ ] Brand colors match the palette tokens exactly.
- [ ] All 3 brand fonts embedded.
- [ ] Spell-check passed.
- [ ] 5-second test passed for every page (a tired parent at 9pm knows what to do with the page in 5 seconds).
- [ ] Greyscale test passed (B&W print is still legible — preps the file for KDP adaptation).
- [ ] All checkbox sizes are uniform.
- [ ] All fillable lines use the gray rule, not black.
- [ ] All icons are from the same icon family.
- [ ] Cover and closing pages have brand URL.
- [ ] Export 1: print-quality PDF (300 DPI, CMYK-safe, fonts embedded). File: `adhd-bts-bundle-print.pdf`
- [ ] Export 2: 11 standalone PDFs (one per printable) for mini-product Payhip listings. Filenames: `adhd-bts-01-morning-routine.pdf` … `adhd-bts-11-goal-setting.pdf`
- [ ] Export 3: 3-page free preview lead magnet (Cover + Morning Routine + Closing Letter). File: `adhd-bts-bundle-preview.pdf`
- [ ] All 5 mockups rendered at 1200×1200 px (square) and 1000×1500 px (Pinterest vertical).

> **When done:** Move to `03-payhip-listing-and-platforms.md` and publish.
