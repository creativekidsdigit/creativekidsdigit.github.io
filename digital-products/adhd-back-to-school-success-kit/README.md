# ADHD Back-to-School Success Kit — Product Factory

A complete digital-product system for ADHD families preparing for the school year — built so a solo creator can ship it in 10 working days and sustain it with ~5 hours/week.

**Priority product:** ADHD Back-to-School Success Kit (26-page printable PDF).
**Audience:** Parents of ADHD kids ages 6–16.
**Hero offer:** $19 launch → $29 standard → $34 holiday. Bundle SKU at $39 launch / $49 standard.

## How to use this folder

**If you have 7 days before launch:** Read `01-product-outline.md` then jump to `06-marketing-plan-pinterest.md` §9 ("minimum viable launch"). Ship the kit + 5 pins + 1 blog post. Layer in the rest weekly.

**If you have 4+ weeks:** Run the full `07-90-day-commercialization-roadmap.md` Phase 0 → 1 → 2.

**If you're starting your *next* product:** Copy this folder, rename it, and follow `08-production-workflow.md` §13 (clone-and-replace map). The factory is reusable.

## Constraint check (from the original brief)

- ✅ **Created and sold quickly by a solo creator.** Total build time: ~54 hours over 4 weeks.
- ✅ **Minimal ongoing maintenance.** Steady-state target: ~5 hours/week by Day 90.
- ✅ **Fits the existing brand.** Same voice, palette, and structure as the existing ADHD toolkits in `/digital-products/`.
- ✅ **Reusable factory.** Every product after this one starts by cloning this folder.
