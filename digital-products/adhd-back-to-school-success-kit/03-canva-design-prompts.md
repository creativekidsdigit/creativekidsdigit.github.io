# 03 — Canva Design Prompts (ADHD Back-to-School Success Kit)

**Doc purpose:** Copy-paste-ready prompts for designing every page of the kit in Canva (or Adobe Express, Affinity Publisher, or any AI-assisted layout tool). Each prompt is structured the same way so a solo creator can move through 26 pages in 1–3 working days.

**How to use this file:**
1. Set up the master file once using §A "Master file setup."
2. For each page, paste the corresponding prompt into your tool's instructions / AI prompt / brief field.
3. Drop the matching text from `02-printable-content.md` into the layout.
4. Export at 300 DPI PDF Print quality.

---

## A. Master file setup

**Document setup:**
- Format: US Letter, 8.5 × 11 in
- Bleed: 0.125 in
- Safe area: 0.75 in margins
- Color profile: CMYK-safe palette (works in print and screen)

**Brand color palette (hex):**
- Primary sage `#A8C5A8`
- Cream `#FAF6EE`
- Terracotta `#D49A7C`
- Butter yellow `#E8D88A`
- Charcoal text `#3A3A3A`
- Soft gray rules `#E0DCD2`

**Typography:**
- Display / headings: **Fraunces** (700 / 500)
- Body: **Inter** (400 / 600)
- Kid-facing handwritten accent: **Caveat** (500)

**Reusable elements to build first (master kit):**
- Page header bar (sage strip with cream title block)
- Page footer (small brand mark + page number)
- 4 callout box styles: sage tinted, terracotta tinted, butter highlight, cream framed
- 3 list-icon styles (filled circle, checkbox, soft dot)
- Cut-line pattern (dashed gray, with scissors icon)
- Cover frame template (full-bleed sage + cream panel)
- Letter template (cream paper, sage thin border)

> **Rule:** Build these once. Every page reuses them. This is what makes the system reproducible for future toolkits.

## I. Production checklist

Run before exporting the final PDF.

- [ ] All 26 pages are present in the correct order.
- [ ] Page numbers in footer are correct and consistent.
- [ ] No leftover Lorem Ipsum / placeholder text.
- [ ] Brand colors match palette exactly (no near-duplicates).
- [ ] All 3 brand fonts embedded.
- [ ] Spell-check passed.
- [ ] Reading order test: read pages 1, 4, 13, 16, 18 cold and check that each makes sense as a standalone print-out.
- [ ] All "[bracket]" fill-in fields are rendered in terracotta and clearly editable.
- [ ] Cut-apart cards on page 14 actually cut to a usable size when printed.
- [ ] Tracker grids on pages 8 and 22 print legibly at 100% scale.
- [ ] Footer credit line + brand URL appear on cover and closing.
- [ ] Export 1: print-quality PDF, 300 DPI, CMYK-safe, all fonts embedded.
- [ ] Export 2: web-preview PDF, ~150 DPI, smaller file for Etsy preview thumbnails.
- [ ] Export 3: 5-page free preview lead-magnet (Cover + Page 4 calendar + Page 13 first-day visual + Page 16 decompression dial + Page 26 letter).
